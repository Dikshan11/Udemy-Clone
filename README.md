# Udemy-Clone
🎓 Udemy Clone (HTML + CSS)
A simple clone of the Udemy homepage created with HTML and CSS.
This project is intended to practice layout, styling, and responsiveness during the first week of learning web development.

🎯Learning Objectives :
👉 Apply CSS for styling and layout
👉 Utilize Flexbox and/or CSS Grid
👉 Create a responsive navbar
👉 Style buttons, course cards, and sections

🚀Features :
✅ Responsive Navbar
✅ Hero section (search bar + categories)
✅ Course Cards Section
✅ Categories Menu
✅ Footer with links and social icons
✅ Hover effects and basic transitions

📁 Technologies Used :
🔷HTML5
🔶CSS3
Google Fonts
Font Awesome
